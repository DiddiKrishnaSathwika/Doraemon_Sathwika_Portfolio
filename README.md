# 🚀 Doraemon-Themed Portfolio

A creative and interactive portfolio website inspired by **Doraemon's futuristic gadgets**, built using **React** and **Vite**. The portfolio showcases my education, skills, projects, certifications, achievements, and contact information through a fun and engaging user experience.

## 🌐 Live Demo

🔗 **Portfolio Website:**
[Doraemon Portfolio Live Demo](https://doraemon-sathwika-portfolio.vercel.app/)

## ✨ Features

* 🎨 Doraemon-themed user interface
* 🚪 Interactive "Doraemon Pocket" navigation
* 👩‍🎓 Education section
* 💻 Skills showcase
* 📂 Projects portfolio
* 🏆 Achievements section
* 📜 Certifications display
* 💼 Experience section
* 📞 Contact information
* 📱 Responsive design
* ⚡ Fast performance with Vite

## 🚀 Installation & Setup

Clone the repository:

```bash
git clone https://github.com/DiddiKrishnaSathwika/Doraemon_Sathwika_Portfolio.git
```

Navigate to the project folder:

```bash
cd Doraemon_Sathwika_Portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
